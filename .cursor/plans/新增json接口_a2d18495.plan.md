# 新增返回 JSON 的接口

## 目标

- 保留现有 `GET /ping` 返回字符串 `pong`
- 新增 `GET /ping-json` 返回 JSON

## 改动点

- 修改控制器文件：[`/Users/yongsheng.hang/luckin/java-demo/src/main/java/com/luckin/javademo/PingController.java`](/Users/yongsheng.hang/luckin/java-demo/src/main/java/com/luckin/javademo/PingController.java)

## 实现方式（最简）

- 在 `PingController` 中新增一个方法，使用 `@GetMapping("/ping-json")`
- 返回类型用 `Map<String, Object>`（避免新增 DTO 类），返回 `{"message":"pong"}`
